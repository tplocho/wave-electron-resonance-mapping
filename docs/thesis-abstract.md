# Thesis Abstract

This thesis examines the wave-electron resonance condition in magnetically confined tokamak plasmas with the aim of building a computationally efficient and physically grounded reference framework for electron-cyclotron heating and current-drive studies.

The work develops a parametric approach to resonance mapping under prescribed tokamak geometry and plasma-profile assumptions, with emphasis on how control variables affect the location and structure of resonant interaction. The resulting framework is intended to support control-oriented reasoning in scenarios related to localized power deposition and instability mitigation, particularly in the context of ECRH/ECCD applications and neoclassical tearing mode control.

The code released in this repository corresponds to the thesis-stage software layer used for these resonance-mapping studies.
